EEPROM Overview

This API contains the functions to configure and control the EEPROM module of the Programmable XBee S2B/S2C devices. The EEPROM module has the following sub-modules:

  • Virtual EEPROM
  • 24xxx EEPROM
  • 25AAxxxx/25LCxxxx EEPROM
